# Prime Properties Analytics Platform 🏢📊

A cloud-based Data Warehousing and Business Intelligence solution developed for the Real Estate & Property Management industry. The project focuses on transforming scattered transactional data into a centralized analytics platform capable of delivering real-time insights, automated reporting, and interactive visualizations.

Prime Properties operates across Karachi, Lahore, and Islamabad, managing residential and commercial property sales and rentals. Previously, data was stored in separate Excel files and legacy systems, which made reporting slow, inconsistent, and difficult to manage.

To solve these challenges, a centralized Data Warehouse system was designed using Snowflake Cloud technology along with modern analytics and visualization tools.

- Track sales and agent performance in real time
